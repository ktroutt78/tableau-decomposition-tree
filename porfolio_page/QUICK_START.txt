QUICK START - View Your Portfolio Offline
=========================================

METHOD 1: Simple Python Server (Recommended for Mac)
-----------------------------------------------------

1. Open Terminal

2. Navigate to the portfolio folder:
   cd "/Users/keithtroutt/Documents/my-claude-project/porfolio_page"

3. Start a local server:
   python3 -m http.server 8000

4. Open your browser and go to:
   http://localhost:8000

5. To stop the server: Press Ctrl+C in Terminal


METHOD 2: Double-Click (May have limitations)
----------------------------------------------

Simply double-click the "index.html" file.
It will open in your default browser.

Note: Some features (like portfolio projects loading)
may not work without a server due to browser security.
